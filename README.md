# Covanix Clean - Nettoyage Professionnel

Bienvenue sur le projet Covanix Clean.

## Informations sur le projet

Ce projet est une application web pour les services de nettoyage professionnel au Maroc.

## Technologies utilisées

Ce projet est construit avec :

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Installation et démarrage

Pour travailler localement sur ce projet, suivez ces étapes :

```sh
# Étape 1 : Cloner le dépôt
git clone <VOTRE_URL_GIT>

# Étape 2 : Naviguer dans le répertoire du projet
cd sparkleclean-solutions

# Étape 3 : Installer les dépendances
npm install

# Étape 4 : Démarrer le serveur de développement
npm run dev
```
